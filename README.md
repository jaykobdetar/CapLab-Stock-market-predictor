# CapLab-Stock-market-predictor
Attempts to predict the stock market 1 day in advance on capitalism lab V11.1.02 does not work on beta branch, no idea on older versions.

## Accuracy

~90–95% accuracy on default settings (base game / custom scenarios)
Not accurate in DLC scenarios or with certain dlc enabled such as the Banking and Finance dlc which enables a global stock market which this does not work with.

## Notes

This works best when starting it from a fresh game.
The accuracy of the player stock and some others stop being reliable after a while, but most of them remain stable.
